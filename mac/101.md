>> plugins=(

  colorize

  zsh-autosuggestions

```shell
echo 'eval "$(starship init zsh)"' >> ~/.zshrc

```

### Git

Install git cmdline tools
```
brew install git tig git-lfs
```

Config for github
```
ssh-keygen -t rsa -C <Work Email>
ssh-keygen -t rsa -C xingled@gmail.com -f ~/.ssh/id_rsa_me
```

Edit `~/.ssh/config` to set different ssh key.
```
Host github.com
  HostName github.com
  IdentityFile ~/.ssh/id_rsa
  PreferredAuthentications publickey

Host gitme.com
  HostName github.com
  IdentityFile ~/.ssh/id_rsa_me
  PreferredAuthentications publickey
```

> - Config git according to `../git`
> - Remember to change `github.com` to `gitme.com` for personal repo
> - Also change `user.name` and `user.email` for local config

### Dev env

#### Node

```
brew install nvm
```

Add to .zshrc:
```
export NVM_DIR="$HOME/.nvm"
[ -s "/opt/homebrew/opt/nvm/nvm.sh" ] && \. "/opt/homebrew/opt/nvm/nvm.sh"  # This loads nvm
[ -s "/opt/homebrew/opt/nvm/etc/bash_completion.d/nvm" ] && \. "/opt/homebrew/opt/nvm/etc/bash_completion.d/nvm"  # This loads nvm bash_completion
```

```
nvm install node
npm install -g npm-check
npm install -g typescript
```

### Docker

##### Install

```shell
brew install hyperkit minikube docker docker-compose
```

##### Config

```
echo "`minikube ip` docker.local" | sudo tee -a /etc/hosts >: /dev/null
echo "eval $(minikube -p minikube docker-env)" >> ~/.zshrc
```

##### Run

```
minikube start -d --no-kubernetes
minikube stop
```
